# Jin-Engine

![](./logo.png)

Jin Engine is a private rendering engine which is based on DirectX 11. (This project is still under development since Aug. 2016)
The goal of this project is to broaden knowledge what what I have studied and to refine the State-of-the-art computer graphics techniques in the future.

Key features

* Physically-based rendering (deferred)
* CubeMap Mip for Global Illumination
* Light Bloom
* Screen Space Reflection (Working…)
* Lens flare
* Depth of Field
* HBAO
* Anti-Aliasing (Working…)
* Shadow Map (Working…)
	- LogPSM
	- Cascade Shadow
	- Sample Distribution Shadow Maps
	- Variance Shadow Maps
